# منصة الدعم النفسي و الإجتماعي – خطوة نحو التعافي

## فكرة الموقع (Problem / Solution)
### المشكلة:
يعاني الكثير من الأفراد، وخاصة المتأثرين بالحرب، من ضغوط نفسية مثل القلق، الخوف، التوتر، واضطرابات ما بعد الصدمة، مع قلة الوعي بطرق الدعم النفسي الصحيحة وصعوبة الوصول إلى الإرشاد المناسب.

## الحل:
تقدّم منصة **خطوة نحو التعافي** بيئة رقمية داعمة تهدف إلى تعزيز الصمود النفسي من خلال التوعية، الإرشاد النفسي والاجتماعي، وتوفير وسائل بسيطة تساعد المستخدم على التعامل مع آثار الحرب بشكل صحي وآمن.

---

## المستخدم المستهدف
- الشباب المتأثرون نفسيًا بالحرب  
- العائلات التي تعاني من ضغوط نفسية  
- الأطفال الذين يحتاجون إلى دعم نفسي وشعور بالأمان  
- أي شخص يبحث عن إرشاد نفسي واجتماعي

---

## أهم الميزات
- عرض خدمات الدعم النفسي والاجتماعي بشكل مبسّط
- نموذج حجز جلسة دعم مع التحقق من البيانات باستخدام JavaScript
- قسم إحصائيات تفاعلي مع خاصية الفلترة باستخدام jQuery
- قسم أسئلة شائعة (Accordion) باستخدام Bootstrap
- عرض فيديو توعوي لدعم المحتوى البصري
- تصميم متجاوب (Responsive) يعمل على مختلف الشاشات

---

## التقنيات المستخدمة
- HTML5  
- CSS3  
- JavaScript  
- Bootstrap 5  
- jQuery  
- Font Awesome
